# Prompt 2: Competitive Analysis with Playwright

Let's use Playwright to have Claude browse competitor websites, take screenshots, and analyze what they're doing.

Copy and paste the following into Claude Code:

```
Context: I have the Playwright plugin installed. I'm opening a boutique fitness studio called "Pulse & Flow" in Austin, TX that combines HIIT training with yoga recovery. Before I launch, I want to study what the top fitness studio brands are doing online so I can stand out.

Instruction: Using the Playwright browser, visit the following fitness studio websites one by one. For each site:
1. Navigate to their homepage
2. Take a full-page screenshot
3. Note their hero message, color scheme, call-to-action, and overall vibe
4. Check if they have a class schedule or pricing page — if yes, screenshot that too
5. Check if they have an Austin, TX location page — if yes, screenshot that too

Then create a competitive analysis summary comparing all three, with:
- What each does well
- What each is missing
- Common patterns across all three
- 5 specific things Pulse & Flow should do differently to stand out in Austin

Input:
- Competitor 1: https://www.barrys.com (Barry's — high-energy group fitness)
- Competitor 2: https://www.purebarre.com (Pure Barre — barre fitness)
- My location: Austin, TX

Output: A competitive analysis with screenshots saved locally and a clear written summary with actionable recommendations for my Austin market.
```

> What you'll see: Claude opens a browser on your machine, navigates to each website, takes screenshots, reads the content, and comes back with a structured analysis. You're watching AI do market research in real time.

> Replace these URLs with competitors in YOUR industry — this works for any business, any niche.
